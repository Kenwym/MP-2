Mon mini projet 1

Partie 1

Le mini projet était par groupe de 2. Avant de commencer chaque code, nous avons crée un menu permettant de choisir ce que l'on voulait faire Dans une première partie, nous devions coder en python un code qui permet de crypter un texte en majuscules via un code césar. L'utilisateur devait entrer le texte à crypter ainsi que la clé de décryptage. Le fonctionnement est illustré par la photo si-dessous :

Partie 2

Dans une deuxième partie, nous devions coder en python un décryptage de texte via un code césar. L'utilisateur avait juste à renseigner le texte à décrypter ainsi que la clé.

Partie 3

Dans une troisième et dernière partie, nous devions coder en python un décryptage en via un code césar, mais cette fois si sans connaitre la clé. L'utilisateur devait rentrer le texte à décrypter. A chaque décryptage effectuer par l'ordinateur, il devait valider ou non le texte décrypter, au bout de 25 essai, (le tour de l'alphabet) si l'utilisateur n'avait pas reconnu de texte, le programme revenait au menu.
